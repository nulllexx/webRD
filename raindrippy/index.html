Solutions

We're working on general, free-to-use, open-source solutions for everybody. Here's a list of what we have to offer:

  1. ddosprot

    A general-purpose Linux & Windows tool made to protect servers from DoS / DDoS attacks.

    How it works:

    ddosprot uses libpcap to monitor incoming traffic on your selected protected ports, and blocks suspicious activity. Inspired by fail2ban, the system uses firewall blocking to discard requests from suspicious IPs.

    Features:

    • Real-time traffic monitoring
    • Automatic IP blocking
    • Customizable protection settings
    • Daemon setup
    • CLI & conf configuration
    • Cross-platform support (Linux & Windows)

    Installation:

    To install ddosprot, download the zip and extract it.

    On Linux, open the Terminal in the extracted folder, and run makepkg -si. On Windows, open PowerShell in the extracted folder, and build the C file with gcc -O2 -Wall -o ddosprot.exe ddosprot.c -I "(path to Npcap SDK)/Include" -I "(path to Npcap SDK)/Lib" -lwpcap -lPacket -lws2_32.
